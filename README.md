## 项目介绍｜Project Introduction

**中文：**

医简（DocEase）是一款面向临床医生的医疗文书辅助软件，旨在通过智能化工具简化病历书写、医嘱管理等繁琐的文书工作，让医生能够将更多精力投入到患者诊疗中。

**核心理念：**
- **医学 + 简单** = 专业医疗与极简操作的完美结合
- **效率优先** — 减少重复劳动，提升文书规范性和完整性
- **医生为本** — 从临床实际需求出发，贴合真实工作场景

**English:**

DocEase is a medical documentation assistance software designed for clinicians. It aims to simplify tedious paperwork such as medical record writing and order management through intelligent tools, allowing doctors to focus more on patient diagnosis and treatment.

**Core Values:**
- **Doc(tor)/Doc(ument) + Ease** = Perfect combination of professional healthcare and minimalist operation
- **Efficiency First** — Reduce repetitive work, improve documentation standards and completeness
- **Doctor-Centric** — Designed from real clinical needs, fitting authentic workflow scenarios

## 预期功能｜Planned Features

| 功能模块 | 功能描述 | Feature Module | Description |
|---------|---------|---------------|-------------|
| 智能模板 | 基于科室和病种自动生成病历模板 | Smart Templates | Auto-generate medical record templates based on department and disease type |
| 语音录入 | 支持语音识别快速录入病历内容 | Voice Input | Support voice recognition for rapid medical record entry |
| 智能补全 | 医学术语智能提示与自动补全 | Smart Completion | Intelligent medical terminology suggestions and auto-completion |
| 质控检查 | 自动检查病历完整性和规范性 | Quality Control | Automated checking of medical record completeness and compliance |
| 专科支持 | 覆盖内科、外科、急诊等多科室 | Specialty Support | Coverage of internal medicine, surgery, emergency, and other departments |
| 数据安全 | 严格遵循医疗数据隐私保护规范 | Data Security | Strict adherence to medical data privacy protection standards |
